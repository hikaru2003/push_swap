# push_swap
42Tokyo LEVEL : 2

# 実行方法
1. make : 実行ファイル sort 作成
2. ./sort data.csv : 引数にcsvファイルを与える
3. ソートアルゴリズムを選択する

ソート中だというメッセージが出る
```bash
now sorting...
```

ソートが完了するとソートされているか、比較回数、実行時間が表示される。

```bash
<result>
sorted!!
count = 186732
time = 1.464353
```
昇順にソートできていれば
```bash
sorted!!
```
できていなければ
```bash
not sorted...
```
と表示される。
